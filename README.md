# ISW CPU

This class is about Web Services and Infrastructures.

## About the work

This site was made in C# and joys with the processes, and permissions from Ubuntu Linux. **The site is made to work only at Ubuntu OS.**

## What you can do with this repos

You can see what process are running along with their PID (Process Identifier) and you can kill any process just by typing it's PID.
You can also run a top command from ubuntu to show how many of yout CPU and RAM the System are using, but that isn't dynamic here. It just shows a copy of one register of the type.
From permission, you can see all of your users and directories, navigate by your directories and change the user who owns the selected directory using the **chown** command.

**It needs your user _password_ to do it**
